# Get Started in 5 Minutes

1. Go to the website folder:

```sh
# Go to the folder
$ cd website
```

2. Make sure all the dependencies for the website are installed:

```sh
# Install dependencies
$ npm i
```

3. Run your dev server:

```sh
# Start the site
$ npm start
```
